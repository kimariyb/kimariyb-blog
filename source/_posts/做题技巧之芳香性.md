## 前言

同学们，大家好，今天来介绍一个考研上必考，但是又比较基础的问题—**芳香性**。那么什么是芳香性呢，根据[百度百科](https://baike.baidu.com/item/%E8%8A%B3%E9%A6%99%E6%80%A7/10882062)的解释，**_芳香性是指在化学性质上表现为易进行亲电取代反应，不易进行加成反应和氧化反应。芳香性的特征是环状闭合共轭体系 π 电子高度离域，具有离域能，体系能量低，较稳定。_**

那么我们如何根据一个化学结构，来推测该物质有没有芳香性呢。在本科的学习中，我们一般使用**休克尔规则**，来讨论一个物质有没有芳香性。什么是休克尔规则我们后面介绍。需要注意的是，休克尔规则并不是所有物质都适用，请不要用休克尔规则解释不能解释的物质。

## 休克尔规则 Hückel Rule

{% note success %}

**_Hückel Rule_**：**Planar, Cyclized, Conjugated System contain 4n + 2 π electrons → Aromaticity**

休克尔规则：平面，单环，共轭的系统包含 4n + 2 的 π 电子，就存在芳香性。

{% endnote %}

> 芳香性的本质其实是，成键/成键加非键轨道全充满。

## 芳香性的判断

### 苯和苯的同系物

有句话说，**存在苯基，就一定有芳香性**，大家可以去思考这句话说得对不对，我这里不说多了。在这里同系物是一个广义的同系物，它是指经典的全碳芳香族化合物，稠环芳烃也有部分属于其中。下面这几种物质，均满足休克尔规则，因此具有芳香性。

![苯和苯的同系物](/img/skill04/skill04-01.png)

### 轮烯的芳香性

轮烯是具有多个共扼双键或叁键的单环共扼烯烃。我们常见的轮烯是苯，苯是有芳香性的物质，这是毋庸置疑的。但是其他的轮烯是否有芳香性呢？我们举一些例子来看看。

![碳数较低的轮烯](/img/skill04/skill04-02.png)

{% note success %}

在上面这些分子中，**都不存在芳香性**。A 的 π 电子是 4 不满足休克尔规则。而 B、C 上的 p 轨道都存在明显的位阻，以至于它们不能很好的平行，从而离域到对方的 p 轨道中，形成不了闭合的环电流。细心的同学可能知道，**_B 和 C 本质上是一种东西_**。

如果将这些环继续扩大成下面的轮烯，那么空间位阻的影响会因为环的扩大而减小，从而具有芳香性。下面物质是具有芳香性的。

![碳数较多的轮烯](/img/skill04/skill04-03.png)

{% endnote %}

### 共轭离子

除了中性的环以外，平面环还可以通过得失 π 电子来满足芳香性规则，从而获得更好的稳定性。这可以用来判断酸性的强弱，如果一个环状结构失去一个质子，形成芳香性的负离子，就说明它的酸性是比较强的。下面是一些常见的具有芳香性的共轭离子。

![共轭离子](/img/skill04/skill04-05.png)

{% note success %}

为什么苯负离子会具有芳香性呢？这是因为虽然苯中有一个碳获得了一个电子，但是这个电子并没有在垂直于苯环平面的 p 轨道上，而是平行的。所以就没有参加共轭，因此实际上还是苯上的六个 π 电子形成了共轭体系，但是由于它带了电荷，稳定性肯定是不如苯环的。

{% endnote %}

## 总结

我们介绍了几种，平常碰见比较多的芳香性物质的判断。当然芳香性的类型还远不止这么点，同芳香性，莫比乌斯芳香性都没有介绍。主要是因为这些内容不常见。

我们现阶段判断芳香性通过休克尔规则就能够判断出来了。大家在判断杂环化合物和离子共轭时，最好把原子轨道画出来，这有利于判断是否拥有芳香性。对于稠环化合物，我们数电子就没问题了。如果大家对文章有什么建议，请联系我。
