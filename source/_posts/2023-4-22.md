# 前言

研究表明，cyclobuta[1,2:3,4]di[5]annulene分子的结构非常特殊。由于其环的对称性和双键构型，许多学者认为它可能具有芳香性。然而，这种化合物是否属于芳香族在学术界仍存在争议。有学者认为该分子具有芳香性，因为其分子平面非常平坦，满足Huckel规则；而另一些学者则认为该化合物不具芳香性，因为其芳香环上没有循环共面的pi电子。

因此，对cyclobuta[1,2:3,4]di[5]annulene的芳香性进行明确的鉴定对于当前研究具有重要意义。本研究使用理论和计算化学工具Gaussian和Multiwfn，探索了该化合物的芳香性。

# 衡量芳香性的方法

## 什么是芳香性

芳香性是化学中重要的概念，虽然其含义通常不够清晰。它通常指含有苯环或类似结构的化合物所具有的特殊化学和物理特性。这些化合物具有独特的气味和味道，以及一定程度的稳定性和溶解性。Huckle规则是测量芳香性使用最久的指标，但越来越多的实验表明其显著的局限性。

现在，有数十个指标用于测量芳香性，并且新的指标不断涌现。这些指标反映芳香性的不同方面，但大多依赖于量子化学计算。由于芳香性没有明确且唯一的定义，该术语涵盖的范围越来越广，因此很难提供简单、准确并被广泛接受的定义。芳香物质通常具有反应性，通过分子中电子的共振构来吸收紫外和可见光并表现出稳定性。

对于大多数芳香体系，它们通常同时具有多个特征，例如平衡键长、电子共振、在外部磁场下产生的环状感应电流、更高的电子共振能量、结构稳定性等。因此，为了更好地描述芳香性，有必要综合考虑基于分子不同性质的多个芳香性指标。下面的章节将介绍本文中用于测量芳香性的指标。

## NICS(0) 和 NICS(1)_zz

NICS是核独立化学位移的缩写，是用于测量芳香性的指标。它最初被定义为环中重原子几何中心处的各向同性化学屏蔽的负值。NICS 指标被提出是为了补充Huckle规则及其变体的不足之处。NICS指标来自于化学位移(CS)的计算模拟，其中可以通过在特定位置引入“测试”核来获得核独立化学位移(NICS) 指标。

NICS可用于计算含有一个或多个芳香环的化合物的芳香性，并确定不同环之间芳香性的差异。对于芳香系统，各向同性化学屏蔽通常为正值，NICS值为负值。以下代码将对已经进行过的几何优化的cyclobuta[1,2:3,4]di[5]annulene分子进行NICS的计算：

## AICD

除了NICS值，AICD也是衡量芳香性的重要指标。AICD（Anisotropy of the Induced Current Density）表示分子局部电荷密度沿不同方向偏离平面的程度。在芳香化合物中，它们的AICD值较低，因为芳香电流沿着环结构中心下沉，导致局部芳香电荷密度降低。

<img src="/image/2023-4-22/2.png" width=55% height=55%>

AICD程序旨在绘制AICD轮廓和感应电流向量，其安装过程在Sobereva的文章中有详细的说明。以下所列的代码将添加到优化的分子文件（gjf）中：

```gjf
# nmr=csgt b3lyp/6-31+g(d) IOp(10/93=1)

cyclobuta

0 1
原子坐标省略

test.txt 
```

使用Gaussian 16执行此文件，得到cyclobuta.out和test.txt。（上文中计算NICS的命令也与此类似）

```bash
	g16 <cyclobuta.gjf |tee cyclobuta.out
```

由于得到的分子几乎是垂直于z轴，即外磁场矢量应该是0 0 1。在终端上输入：

```bash
	 AICD -m 4 -b 0 0 1 -pov cyclobuta.out
```

将得到的RenderMich.pov文件使用Povray渲染器打开并进行渲染，得到下图：

生成的图像清晰地显示cyclobuta[1,2:3,4]di[5]annulene中环电流的存在。此外，Sobereva在其文章中还提到了使用AICD生成只含某特定轨道的磁感应电流图的方法。为了获取分子中的所有pi轨道，使用Multiwfn程序搜索所有pi分子轨道。并修改上述代码：

用上文操作继续进行绘图，得到的结果如下图所示：

通过比较只对pi轨道和所有分子轨道的磁感应电流图，我们观察到了结果上的差异。对pi轨道产生的磁感应电流图呈现出明显的中断，而完整的磁感应电流图则清晰的表现出环电流的存在。因此使用AICD来判断cyclobuta[1,2:3,4]di[5]annulene的芳香性并不理想。

## HOMA

HOMA (Harmonic oscillator measure of aromaticity)指数用于衡量分子的芳香性，其基于计算分子中碳-碳键的芳香程度来确定其整体芳香性。指数通过计算pi电子在形成环结构时相对于其简单偶极矩平衡位置的偏离程度来评估芳香性的程度。HOMA值范围为0至1，其中0表示非芳香性，而1表示完全芳香性。标准HOMA值提供定量信息，可用于比较不同分子的相对芳香性。HOMA指数的表达式如下：

## ELF-sigma/pi 和 LOL-pi

电子定域化函数（Electron localization function，ELF）是用于评估分子中电子局域性的一种指标。ELF通过测量一个点周围的电子定域程度来评估化学键的性质和局域性。

ELF-sigma和ELF-pi用于研究sigma和pi芳香性。随着等值面的数值（isovalue）增加，ELF等值面内的空间（称为域）会逐渐变小。如果域内含有多个ELF极大点（叫可约域），则随着isovalue的增加，这个连通的域就会分解为多个小的孤立的域，最终每个域内只包含一个ELF极大点（叫不可约域）。

一般认为，ELF-pi大于0.7的体系可以算作具有pi芳香性，而在0.11~0.35之间则可以算作有pi反芳香性。对于有机分子，ELF-pi和ELF-sigma的平均值大于0.7可以算作具有整体芳香性，小于0.55的体系则可以算作具有整体反芳香性。

## ICSS

ICSS (Iso-Chemical Shielding Surfaces) 是研究立体位阻效应和非平面效应的工具。它被定义为沿化学键等化学位移面的等值面。与传统计算在环中心处 NICS值相比，这种方法更加直观并显示了更丰富的信息。

类似地，Sobereva 在他的文章中介绍了使用Multiwfn计算ICSS_zz的方法。值得注意的是，ICSS和NICS的符号是相反的。NICS 取负的磁化位移值，这意味着 NICS 值越负数，外部磁场对该点的屏蔽效应就越强。NICS值越正，表示该位置的非磁性屏蔽效应越强。另一方面，ICSS直接显示磁性屏蔽值，无需取负值。

与 NICS 相似，ICSS_zz比ICSS更具有可信度，如下图中，绿色等值线表示屏蔽区为2 ppm的区域，而蓝色等值线表示去屏蔽区为-2 ppm或屏蔽区为2 ppm的区域。

<img src="/image/2023-4-22/ICSSzz.png" width=70% height=70%>

从cyclobuta[1,2:3,4]di[5]annulene的ICSS_zz等值面图可以看出，绿色部分位于环内，表明发生了屏蔽效应。周围是蓝色部分，进一步表明了存在环外去屏蔽区域。因此，该分子存在环电流，从而推断出该分子具有芳香性。 

# 总结

本研究采用理论和计算化学方法分析cyclobuta[1,2:3,4]di[5]annulene的芳香性质。通过LOL-pi填充图和ELF图表明该分子呈现出一定的芳香性。并且计算得到的ICSS_zz等值面图也表明该分子具有环流，并因此呈现出芳香性。
